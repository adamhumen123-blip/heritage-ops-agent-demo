# Client Demo Walkthrough

## 90-second presentation

1. Open the live dashboard and explain that it combines order, inventory, finance, customer-care, and marketing signals in one executive view.
2. Click **Run Attention Agent** to refresh the prioritised management briefing.
3. Point out one VIP order issue, one overdue receivable, and one low-stock item to demonstrate cross-functional reasoning.
4. Use **Assign** or **Approve** on an attention item.
5. Show how the item leaves the queue, KPI counts update, and the decision appears in the activity log.
6. Explain that the public demo uses safe sample data. The paid pilot replaces these samples with authenticated Shopify, Xero, helpdesk, inventory, Klaviyo, and Google Workspace connectors.

## Suggested client message

I built this pilot around the first capability I would recommend for your business: a daily attention agent. It consolidates operational data, identifies exceptions, prioritises them commercially, produces an executive digest, and supports controlled actions with an audit trail. The current version uses realistic sample data so you can test it without exposing credentials. In production, I would connect the same workflow to your actual systems and automate approved actions through n8n or Make.
